# 画像品質管理チェックリスト

> **運用ルール**: 生成された全画像は、SNS投稿・Fanvue公開の前に必ずこのチェックリストに基づいて目視確認すること。

---

## 必須チェック項目

### 1. 顔の一貫性 ✅

- [ ] リファレンス画像と比較して同一人物に見えるか
- [ ] 顔のパーツ（目・鼻・口）の位置と形が正しいか
- [ ] 表情がキャラクター設定と合致しているか
- [ ] 目が同じ方向を向いているか（斜視チェック）

### 2. 解剖学的な正確さ ✅

- [ ] 指の本数は正しいか（片手5本）
- [ ] 手の形が自然か（融合・変形がないか）
- [ ] 体のプロポーションが自然か
- [ ] 関節の曲がり方が物理的に正しいか
- [ ] 耳の形と位置が正しいか

### 3. AI感の排除 ✅

- [ ] 肌がプラスチックのように不自然に滑らかでないか
- [ ] 肌のテクスチャ（毛穴・産毛）が適度に見えるか
- [ ] 髪の1本1本が自然に流れているか
- [ ] 目に自然なキャッチライトがあるか
- [ ] 全体的に「AI生成っぽさ」を感じないか

### 4. 衣装と小道具 ✅

- [ ] 衣装の柄やテクスチャが歪んでいないか
- [ ] ボタン・ジッパー・縫い目が自然か
- [ ] アクセサリーの形が正しいか
- [ ] 靴が左右対称か

### 5. 背景と構図 ✅

- [ ] 背景に不自然なオブジェクトや歪みがないか
- [ ] テキスト（看板等）が意味不明な文字列になっていないか
- [ ] 構図がシーン設定と合致しているか
- [ ] ライティングの方向が一貫しているか

### 6. 規約適合性 ✅

- [ ] **free_sns tier**: 完全SFW基準を満たしているか
- [ ] **subscriber tier**: チラリズム/フェチが許容範囲内か
- [ ] **premium tier**: Fanvue規約に違反していないか
- [ ] 未成年に見える要素がないか
- [ ] 実在の人物に似すぎていないか

### 7. 技術品質 ✅

- [ ] 解像度が十分か（最低 1664x2432 @ 2x upscale）
- [ ] ウォーターマーク・テキストの混入がないか
- [ ] JPEG アーティファクトが目立たないか
- [ ] ノイズが過度でないか

---

## 判定基準

| 判定 | 基準 | アクション |
|------|------|-----------|
| ✅ **採用** | 全項目チェック通過 | 素材ライブラリに登録 |
| ⚠️ **要修正** | 1-2項目で軽微な問題 | Inpainting / ADetailerで修正後に再チェック |
| ❌ **不採用** | 3項目以上で問題 or 規約違反 | 破棄して再生成 |

---

## 効率化のコツ

1. **バッチ生成後、まず規約適合性（#6）を確認** — 違反画像は即破棄
2. **次に顔の一貫性（#1）** — リファレンスと明らかに違う画像を排除
3. **残りの画像に対して解剖学的チェック（#2）** — 指・手に問題がある画像を Inpainting で修正
4. **最後にAI感 / 背景 / 技術品質を確認** — 微調整可能なら修正、不可能なら再生成
