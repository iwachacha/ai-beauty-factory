# AI美女収益化プロジェクト (AI Beauty Monetization) コンセプト・設計書

## プロジェクト概要
SNS上で実在するかのようなAIインフルエンサー（AI美女）を運用し、Fanvueなどのプラットフォームでのサブスクリプション収益と、SNS上でのエンゲージメント獲得を自動化・効率化するプロジェクトです。
単なる画像生成にとどまらず、キャラクターの「バックボーン」「生活感」「フェティシズム」を綿密に設計し、ダッシュボードから一元管理できるシステムを構築しています。

## 2. 独自プロンプトエンジン（チラリズムとフェチ機構の統合）
既存のありふれたAI美女アカウントと差別化するための強力な武器として、**「意図しないセクシーさ（隙・ラッキースケベ要素）」**を100%言語化し、プロンプト生成DB（`prompt-blocks.json`）へ組み込みました。

### チラリズム（全18種） - 視覚的・構造的な隙間
1. 絶対領域 / 2. オフショル / 3. 浴衣隙間 / 4. 風スカート / 5. バックライン / 6. クロスレッグ / 7. 太ももの肉感（乗っかり） / 8. 肩紐ずり落ち / 9. 髪を結ぶ仕草 / 10. 谷間・俯瞰 / 11. パンちら / 12. 前かがみ胸ちら / 13. ブラ透け / 14. パンティーライン(VPL) / 15. ボタンの隙間 / 16. あぐら・体操座りの隙間 / 17. 床の物を拾う後ろ姿 / 18. 着替えの途中

### フェティシズム（全16種） - 仕草とリアルな質感
1. 濡れ透け / 2. 汗ばみ / 3. 首筋汗 / 4. タオル巻き / 5. 雨濡れ髪 / 6. 朝ストレッチ / 7. 入浴 / 8. 脇見せ / 9. 鎖骨 / 10. シースルー・レース素材 / 11. ずれた眼鏡 / 12. OLスーツ・パンスト足 / 13. ピタニット（リブニット） / 14. 下着の紐を直す / 15. 太ももに挟んだ手 / 16. 唇についた髪・指で拭う

#### ティア（投稿先）運用ルール
これらの要素は、投稿先のプラットフォームに合わせて段階的に適用します。
- `free_sns` (X, TikTokなど): 完全SFW。ブラ透けやリブニットなど、間接的なフェチを使用。
- `subscriber` (Fanvue無料会員): 濡れ透けや着替えの途中など、より強い隙やフェチを提供。
- `premium` (Fanvue有料会員/PPV): 入浴、タオル巻き、あぐらの隙間など、最もプライベートなコアな領域。

## 3. システムアーキテクチャ
生成品質の一貫性と、SNS運用の効率を最大化するためのエコシステムです。

### A. バックエンド・ダッシュボード（Factory Shell）
- **技術スタック**: Next.js (Turbo) + NestJS + MongoDB
- **主要モジュール**:
  - `Characters`: キャラクター情報のCRUDと各種ステータス。
  - `Templates`: 30種類以上におよぶ「シーンテンプレート（オフィス、カフェ、自宅など）」のDB。
  - `Calendar`: キャラクター別・日別のSNS投稿スケジュール管理。
  - `Monetization`: プラットフォーム（Fanvue等）別収益データのレポーティング（開発中）。

### B. 画像生成パイプライン
- **プラットフォーム**: ComfyUI (RunPod等のクラウドGPU上で稼働想定)
- **顔の一貫性 (IP-Adapter FaceID)**: 「AI美女は写真のたびに顔が変わる」という弱点を克服するため、各キャラごとに高品質な【正面・斜め・横顔】のリファレンス画像を生成して固定。これをIP-Adapterで適用します。
- **バッチ自動化**: PythonによるAPI連携 (`scripts/batch_generate.py`) を実装しており、プロンプトブロックDBを読み込み、ランダムな衣装×フェチ×シナリオの画像を自動で一括並列生成します。

## 4. 品質評価とAI感の排除
リアルな人間と区別がつかないクオリティを目指し、独自の徹底したチェックリスト (`quality-checklist.md`) を導入。
以下の要素を厳しく監査します。
1. **解剖学的正確性**: 瞳孔の歪み、指の数だけでなく、「手首や足首の曲がり方」などの見落としがちな違和感を排除。
2. **肌・ライティング**: 「プラスチックのような過剰に滑らかな肌」をNGとし、毛穴設定やリアルな影（Golden Hour, Soft Morning Light等）を加えます。
3. **服の物理法則**: 胸の形にピタッと張り付く違和感のある衣服ではなく、自然なシワや「リブ感」を追求しています。

本プロジェクトは、**「究極のリアリティ×男が抗えない日常の隙」** をシステマチックに量産することで、長期的なエンゲージメントと収益化を実現する自動化マーケティング基盤です。
